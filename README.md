# 🧠 GenAI Text Summarization Agent

## 🚀 Overview

This project is an AI-powered text summarization agent built as part of the **Google GenAI Academy APAC 2026**.

It uses Generative AI to summarize long-form text into concise and meaningful outputs.

---

## 🛠️ Tech Stack

* Python
* Google Generative AI (Gemini)
* FastAPI / Agent-based architecture
* Google Cloud Run (Deployment)

---

## ⚙️ Features

* Summarizes large text inputs
* Uses LLM-based reasoning
* Scalable deployment using Cloud Run
* Modular agent design

---

## 📦 Project Structure

```
.
├── agent.py
├── requirements.txt
├── __init__.py
```

---

## ☁️ Deployment

This application is deployed using **Google Cloud Run**, enabling scalable and serverless execution.

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
python agent.py
```
